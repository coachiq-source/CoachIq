Week 2 Field Sheet
Ground balls and clearing — learning the game under live pressure
Outcome: Every player can execute a clean ground-ball scoop under light pressure, identify their outlet on a clear, and name three rule violations they've seen (and corrected) in practice.
First Game
May 30
Primary Target
Player development
Field
Full, standard
Conditioning
8–10 min dynamic: high knees · butt kicks · carioca · A-skip · backward run · 20y sprint + jog back. Two rounds. Sticks throughout.
Goalie Track
Position unsettled — rotate 3–4 candidates this week. 0–25 min parallel: warm-up clears + save progression. Pick starter Thursday.
Session 1
GB + Stick Skills
0–10: Activation
10–25: Noodle Scooping (entry GB drill)
25–45: Partner Passing + Cradle Ring
45–60: 1v1 GB's (intermediate)
60–72: 3v3 Groundballs live
72–75: Cool-down + message
Decision: ID top 3 GB players by name
Session 2
Clearing Basics
0–10: Activation + outlet pivot drill
10–25: 4v3 Box (entry clear)
25–45: Scoop and Shoot
45–60: 5v4 House (intermediate clear)
60–72: 4v4 Clear to Offense live
72–75: Cool-down + message
Decision: Pick first-line clearing unit (4 players)
Session 3
GB + Transition
0–10: Activation + scoop-to-carry drill
10–25: 2v2 Canada GB's (full GB drill)
25–45: 3v2 West Genny (unsettled offense)
45–60: 4 Corners Pass + motion intro
60–72: 5v5 GBs to Transition live
72–75: Cool-down + message
Decision: Name starting five for Session 4
Session 4
Live Scrimmage + Rules
0–10: Activation + 2×40y sprints
10–25: Ground Ball Gauntlet (all 19 players)
25–40: Clearing review (4v3 + 5v4 mixed)
40–70: Full scrimmage (10v9, 15-min halves, constraint = players call rule violations)
70–75: Cool-down + message
Decision: Name starting goalie for Week 3
Drill Session Key Cue
Noodle Scooping 1 "Butt down, top hand down, scoop through it."
1v1 GB's 1 "Box out with your hips, then scoop away from pressure."
4v3 Box 2 "Goalie looks wing first; defenders spread wide."
5v4 House 2 "If you're covered, swing it back through the goalie."
2v2 Canada GB's 3 "Scoop, turn, find the outlet in one motion."
3v2 West Genny 3 "Numbers up — push the pace, don't let them set."
Tracking Priorities
  • GB win rate: 60%+ by Session 4
  • Clear success: 70%+ reach offensive box
  • Rule calls: 5+ correct calls by players

Watch For
  • Bent-waist scoops (cue "butt down")
  • Goalie clears too late
  • Top hand choked up on plastic
  • Wings don't show hands on clears
  • Offsides confusion in scrimmage
